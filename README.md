# resume-vue

vue imitate [https://github.com/resume/resume.github.com](https://github.com/resume/resume.github.com)

## Project setup

```
yarn install
```

## Compiles and hot-reloads for development

```
yarn run serve
```

## Compiles and minifies for production

```
yarn run build
```
